Tag de Hiperlink

A tag a href define um hiperlink (conexão), que é utilizado para vincular uma página a outra.

O atribulto mais importante desta tag é o href que define a página de destino que será exibida após o lcick domouse no link.

Por padrão, os links antecederão da seguine forma em todos os navegadores:

um link não visitado é sublinhado de azul

Um link visitado é subinhado de roxo

Um link ativo e sublnhado de vermelho

Exemplo

Clique aqui para visitar a página sobre marcadores h1-h6

Link para o site do SENAI Suzano

Clique aqui para visitar a página do SENAI Suzano
